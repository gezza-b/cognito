This is a simple Cognito PoC that shows identity administration via AWS Cognito. Pre-requisites for this PoC are:

  • AWS: you will need to setup a Cognito Idenity Pool
  • Facebook: you need to create an application ID
  • AWS: enable CORS on your S3 bucket

The PoC achieves the following outcomes:

  • anonymous.html: Registers a new anonymous users to the Identity Pool
  • facebook.html: Authenticates a user via Facebook and registers the authenticated user to the Identity Pool
  • register1.html: Captures user information registers a new user to the Cognito User Pool
  • authenticate_up.htmlauthenticates a user and shows the user details
